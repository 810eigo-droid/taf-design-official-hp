## 配信スケジュール(推奨)

| 通 | タイミング | 目的 |
|---|---|---|
| 1通目 | 登録の**翌日** | 資料の読みどころ+「まずは予算」ページ案内 |
| 2通目 | 登録の**4日後** | 事例・お客様の声(安心材料) |
| 3通目 | 登録の**7日後** | クーポン+見積もりへの後押し(クロージング) |

- 資料請求の**直後**はCF7の自動返信(資料URL+LINE案内)が既にあるので、1通目は翌日でOK。
- 3通で完結。しつこくしない(Spam rate 0.1%未満の死守が最優先)。

# 🛠 MailPoetへの設定手順

## 事前確認(1回だけ)

1. **設定 → 基本設定** で以下を確認:
   - デフォルト送信者: `TAF design` / `info@taf-design.com`
   - 「購読解除ページ」等はMailPoet標準のままでOK(配信解除リンクは自動で入る)
2. **フッターの事業者表記(特定電子メール法)**:
   メールのフッターに「送信者名(屋号)+所在地+配信解除リンク」が入る状態にする。
   MailPoetのテンプレートのフッターに屋号と連絡先を記載しておく。

## リストを作る

1. **MailPoet → リスト → 「新しいリストを追加」**
2. 名前: `資料請求者` (説明: offer2フォーム経由の資料請求リード)

## ステップメール(自動化)を作る

1. **MailPoet → 自動化(Automations)→ 「+ 新しい自動化」**
2. テンプレートから **「Welcome new subscribers(新規購読者を歓迎)」** を選ぶ
3. トリガー: **「リスト『資料請求者』に登録されたとき」**
4. ステップを3つ作る:
   - **メール1**: 遅延 **1日後** → 上の1通目をコピペ
   - **メール2**: 遅延 **さらに3日後**(=登録から4日) → 2通目をコピペ
   - **メール3**: 遅延 **さらに3日後**(=登録から7日) → 3通目をコピペ
5. 各メールの件名・プレビューテキストも上記からコピペ
6. **有効化**する前にテスト送信(自分のGmail宛)で表示確認

## 購読者の入れ方(当面の運用)

- **新規**: offer2フォームからの資料請求者を「資料請求者」リストに追加
  (タスク5で、フォームにオプトインチェック+MailPoet連携を検討)
- **既存のFlamingo 8件**: 本人の同意が明確でないため**勝手に追加しない**。
  個別に「メール案内を送ってもいいですか?」と確認が取れた人だけ手動追加
  (MailPoet → 購読者 → 新規追加)
