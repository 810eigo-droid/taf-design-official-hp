## 1. 明日やること(合計 約30分)

### ① Google Search Console 登録(5分・最優先)

1. https://search.google.com/search-console を開く(**810eigo@gmail.com** でログイン)
2. 「**ドメイン**」プロパティを選び `taf-design.com` を入力(https:// は付けない)
3. TXTレコードの画面が出たら **そのまま「確認」を押すだけ**
   (今朝Postmaster用に入れたTXTで即認証が通る。DNS作業は不要)
4. 登録後にやること:
   - 左メニュー「サイトマップ」に `sitemap.xml` を送信(WordPressの場合 `wp-sitemap.xml` も試す)
   - 数日後から「検索パフォーマンス」で、どんな検索語で表示されているかが見え始める

### ② 現状把握(10分)

- Search Console の「検索パフォーマンス」…最初はデータ無しでOK(計器盤を付けたことが大事)
- Googleで実際に検索してみる: `site:taf-design.com` → 何ページがインデックスされているか数える
- 「ティーチ&ファネルDesign」で検索 → 自社名で1位に出るか確認(出なければ屋号の統一表記を要検討)

### ③ 戦略の分岐を決める(5分・要オーナー判断)

**Q: 制作クライアントは「全国(オンライン)」か「地元エリア中心」か?**

- **地元中心** → ローカルSEOを主軸に(コスパ最強の勝ち筋):
  - Googleビジネスプロフィール(無料)に登録: https://business.google.com/
  - 対策キーワードを「地域名 × ホームページ制作」に(例:「〇〇市 ホームページ制作 格安」)
  - サイトのtitle・descriptionにも地域名を追記
- **全国** → 下記のロングテール記事戦略に全振り

## 2. コンテンツ戦略 — 「今日やったことを記事にする」

### なぜこれが勝てるのか

- 2026-07-22に実施したメール到達性の設定(SPF/DKIM/DMARC・Postmaster・Brevo・MailPoet)は、
  **中小事業者が実際に検索して困っている題材**
- 手順書がGitに完成済み → 記事化コストが低い
- 「実際にやった・スクショ付き・結果10/10満点」= 実体験(E-E-A-T)でAI量産記事に勝てる
- 読者=フォームのメールが届かず困っている事業者 = **そのまま見込み客**
