# MailPoet 3通目 コピペ用テキスト版

MailPoetのテキストブロックにそのまま貼り付けるための3通目メールです。

## 件名案

```text
最後に、LINE限定クーポンと1分見積もりのご案内です
```

## プリヘッダー案

```text
資料請求後のご案内は今回で一区切りです。今後は不定期でWeb制作やクーポン情報をお届けします。
```
