# MailPoet 1通目 コピペ用テキスト版

MailPoetで区切り線ブロックが使えない場合は、このままテキストブロックに貼り付けてください。

ボタンが使える場合は、本文中のURL部分をボタンに置き換えてもOKです。

## 件名

```text
資料請求ありがとうございます。まず見てほしい3か所だけお送りします
```

## プリヘッダー

```text
資料は全部読まなくても大丈夫です。まずは料金・制作の流れ・保守管理プランだけ確認してください。
```
