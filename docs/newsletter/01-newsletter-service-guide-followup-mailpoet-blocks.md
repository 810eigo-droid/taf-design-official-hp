# MailPoet 1通目 ブロック作成用

MailPoetは、ニュースレター本文にカスタムHTMLテンプレートをそのまま貼り付ける運用に向いていません。

公式ヘルプでも、MailPoetのメールエディタではカスタムHTMLは自動的に削除されると説明されています。

そのため、このメールはMailPoet上で以下のブロックを順番に追加して作成します。

## 件名

```text
資料請求ありがとうございます。まず見てほしい3か所だけお送りします
```

## プリヘッダー

```text
資料は全部読まなくても大丈夫です。まずは料金・制作の流れ・保守管理プランだけ確認してください。
```

## ブロック構成

### 1. テキストブロック

```text
こんにちは。
ティーチ&ファネルDesignの大村です。

昨日は資料のご請求、ありがとうございました。

「まだ全部読めていない…」という方もご安心ください。
実は、最初に見ていただきたいのは次の3か所だけです。
```

### 2. 区切り線ブロック

MailPoetの「Divider / 区切り線」を追加します。

### 3. テキストブロック

```text
■ 読みどころ 1：料金ページ

「Web制作は高い」というイメージが変わるはずです。
当方は初期費用2万円〜。
理由も資料に書いています。


■ 読みどころ 2：制作の流れ

ご依頼から公開まで、何をするのかが見えると
不安がグッと減ります。


■ 読みどころ 3：保守管理プラン

作って終わりではなく「その後」まで一社完結。
ここが他社との一番の違いです。
```

```text
予算を選ぶだけで、
「残すもの」「後回しにするもの」をプロが設計する
ページをご用意しています。
```

## 注意

MailPoetのフッターにある配信解除リンクは消さないでください。

配信解除リンクと送信者情報は、メルマガ運用上とても大事です。
