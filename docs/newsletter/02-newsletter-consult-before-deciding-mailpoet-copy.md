# MailPoet 2通目 コピペ用テキスト版

MailPoetのテキストブロックにそのまま貼り付けるための2通目メールです。

## 件名案

```text
何を頼めばいいか決まっていなくても大丈夫です
```

## プリヘッダー案

```text
ホームページ制作は、内容を完璧に決めてから相談しなくても大丈夫です。予算から一緒に整理できます。
```
