# 引き継ぎ: /budget/(まずは予算から選ぶ)への導線追加アイデア

> 作成日: 2026-07-19(公式HPセッションで作成)
> 宛先: LP-V2セッション(TAF-V2meta-LP)ほか、資料ページ・LINE等を扱うチャット
> 目的: 新設した予算ハブページ https://taf-design.com/budget/ への導線を各所に張る

## /budget/ とは(前提共有)

- 「まずは予算」ポリシーのハブページ。予算(〜2万/2〜5万/5〜10万/10万以上)を選ぶと
  「残すもの/後回しにするもの」が切り替わるセレクターつき
- 全プラン個別ページへのリンクを持ち、CTAは /order/(見積もり)に接続済み
- ブロック原本: 公式HPリポジトリ `docs/wp-blocks/wp-budget-selector.html`
- 公式HP側の導線は設置済み: トップナビ「まずは予算から選ぶ」/ハンバーガー/フッター料金・プラン列

## リンクルール(全チャット共通)

- 見積もり = `https://taf-design.com/order/`
- 相談・資料 = `https://taf-design.com/offer2/`
- 予算ハブ = `https://taf-design.com/budget/`
- プラン個別: lp-hp2(LP・HP 2万〜)/hp3(WP HP/LP 3万〜)/app2_9(AIアプリ 2.9万〜)/
  lp7(プロ 7万〜)/lp15(ファネル 15万〜)/ad(広告運用 月1.8万〜)/price(料金表)/coupon

## アイデア一覧(優先順位つき)

### A. 資料ページ https://taf-design.com/lp/service-guide.html(最優先)

1. 【優先1】価格・プラン紹介セクションの直後に「まずは予算から選ぶ →」ボタンを設置
   (読んで価格が気になった瞬間に誘導するのが最も効く)
2. 【優先2】資料の最後(読了後のクロージング)に
   「読み終わったら、次は予算を決めるだけです」+ /budget/ ボタン
3. 「まだ何を頼むか決まっていない方へ: 予算だけ決めれば、残すもの・後回しは
   プロが設計します → /budget/」という不安解消の一文
4. 資料内の各サービス紹介箇所に、該当プラン個別ページへの「詳細・料金 →」リンク
   (LP紹介→lp-hp2 / HP紹介→hp3 / AIアプリ紹介→app2_9 / ファネル紹介→lp15 / 広告紹介→ad)
5. (任意・後回し可)画面下に控えめな追従ボタン「💰 予算から選ぶ」

### B. 資料ページ以外

6. 各プランページ(6ページ)から /budget/ への逆リンク:
   「このプランは予算◯万円台の方向け。他の予算帯を見る → /budget/」(回遊づくり)
7. LINE公式のリッチメニューに「予算から選ぶ」ボタン追加(見積もり前の迷い客の受け皿)
8. 見積もりアシスタント(claude.aiプロジェクト)に /budget/ を教え、
   「予算が未定の方はまずこのページへ」と案内させる
9. offer2(お問い合わせV2)に「先に予算だけ決めたい方はこちら → /budget/」
10. クーポンページに「クーポンは5万円(税抜)以上で利用可 → 予算5〜10万円の方はこちら」

## ボタンの見た目(流用可)

公式HPリポジトリ `docs/wp-blocks/wp-quick-estimate-button.html` の
簡単見積もりボタンと同型で、文言・リンク先を差し替えるのが手早い:

- メイン文言: 「まずは予算から選ぶ」
- サブ文言: 「予算を選ぶだけで、残すもの・後回しがわかります」
- リンク先: `https://taf-design.com/budget/`
- 配色ルール: 見積もり系=白地+金枠(アウトライン)。/budget/ は見積もりの前段なので同系でOK

## スマホ優先の導線ルール(重要・全チャット共通)

**同一ドメイン内のリンクに `target="_blank"` を使わないこと。**
スマホユーザーは「ブラウザの戻るボタン」でしか前のページに戻れない人が多い。
別タブで開くと戻るが効かず迷子になるため、同じタブで開いて戻るで帰れるようにする。

- 資料ページ(service-guide.html)内の以下のリンクも同一タブに修正すること:
  料金表(taf-design.com/price/)/制作実績(foriio)/お見積り(taf-design.com/order/)/公式サイト(taf-design.com)
- **PDF表示・印刷ボタンも同様**。PDFを別タブや別アプリで開くと戻れなくなるため、
  同一タブで開くか、ページ内で完結する方法(印刷ダイアログを直接開く等)にする
- 例外: tel: / mailto: / SMS / LINE(lin.ee)など外部アプリを起動するリンクはそのままでよい
