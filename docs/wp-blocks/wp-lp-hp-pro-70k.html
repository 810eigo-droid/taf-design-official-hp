VOICE INPUT

文章作成が苦手な方へ

「まとまった文章を書くのが大変」という方は、スマホの音声入力で問題ありません。 スマホのメモ帳でマイクボタンを押し、サービスへの想いやターゲットへのメッセージを自由に話してください。 文法や誤字脱字は気にしなくて大丈夫です。

変換されたテキストをお送りいただければ、内容を整理し、プロ仕様のキャッチコピーやページ構成に整えます。

1. マイクをタップスマホのキーボードにある音声入力ボタンを押します。
2. 自由に話す想い、強み、お客様に伝えたいことをそのまま話します。
3. 文字を送る出てきた文字をそのままお送りください。こちらで整えます。
