# 既存リード向け・手動フォローメール(1回きり)

> 作成: 2026-07-22 / 対象: Flamingoアドレス帳の既存リード(8〜10件・ステップメール開始前の資料請求者)
> 方針: ステップメール(自動)のリストには入れない(オーナー決定)。
>       代わりに1回だけの手動メールで LINE・見積もり・予算ページを案内する。
> 送信方法: Gmailから差出人 info@taf-design.com で1人ずつ個別送信(BCC一斉はしない)。
>           mail-tester 10/10 確認済みルート。新しいリードから順に。

## 件名

```
先日の資料のご請求ありがとうございました|その後いかがでしょうか
```

## 本文(「〇〇様」だけ差し替え)

```
〇〇様

先日は資料をご請求いただき、ありがとうございました。
ティーチ&ファネルDesignの大村です。

その後、資料はお役に立ちましたでしょうか。
ご不明な点があれば、このメールへの返信で
お気軽にご質問ください。

今日は、ご検討に役立つご案内を3つだけ。

■ ① LINE限定クーポン
公式LINEで、お得なクーポンを不定期に発行しています。
発行は予告なし。登録している方だけが受け取れます。
(税抜5万円以上のご依頼で利用可能です)

▼ LINEで友だち追加(10秒で完了)
https://lin.ee/pqfXTUJ

■ ② 見積もりは1分で終わります
項目を選ぶだけで、金額がその場でわかる自動見積もりです。
こちらから営業のご連絡はしませんので、お気軽にどうぞ。

▼ 1分で自動見積もりを試す
https://taf-design.com/order/

■ ③ 「何を頼むか決まっていない」方へ
先に決めるのは、内容ではなく予算です。
予算を選ぶだけで「残すもの・後回しにするもの」がわかる
ページをご用意しています。

▼ まずは予算から選ぶ
https://taf-design.com/budget/

※今後のご案内が不要な場合は、お手数ですが「不要」とだけ
ご返信ください。以後お送りいたしません。

ティーチ&ファネルDesign 大村
https://taf-design.com/
info@taf-design.com
```

## 送信ルール

- 1人ずつ個別に(宛名を差し替え)。一斉BCCはしない
- 末尾の「不要ならご返信ください」は削らない(受信拒否の案内・特定電子メール法対応)
- 「不要」と返ってきた人には以後一切送らない(Flamingo側にメモ)
- 返信(質問など)が来た人は、会話の流れで「メール案内を続けてもいいですか」と
  確認できれば、同意を得てMailPoetリストに追加してもよい
